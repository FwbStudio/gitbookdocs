# Installation

### **Framework**

* [ESX](https://github.com/esx-framework/esx_core)
* [QBCore](https://github.com/qbcore-framework/qb-core)

### Dependencies

| Resource                                                               | Install  | Description                |
| ---------------------------------------------------------------------- | -------- | -------------------------- |
| [ox\_lib](https://github.com/overextended/ox_lib)                      | Required | context                    |
| [fs\_bridge](../fs_bridge/installation.md) | Required | callbacks and other stuffs |

### 1 - Start

```
1. Download your resource from FiveM's Keymaster.
2. Unzip fs_safezonecreatorr.zip folder and place it into your resource folder.
```

### 3 - Configuration

{% code overflow="wrap" %}

```
* you can add different type of weapons for whitelisting in safezone in config

* You can Configure commands and commands permissions in config.

-- Commands:
* /safezonemenu

* just drag drop it with our other resources or you can check How to Install FS Scripts on top
```

{% endcode %}

### 5 - Ready

Congratulations, you've successfully installed the resource. Restart the server and you will be all set.
