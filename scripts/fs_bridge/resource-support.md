# Resource Support

This page lists the supported resource groups currently exposed through `sh_config`.

## Ambulance

`auto`, `esx_ambulancejob`, `wasabi_ambulance`, `qb-ambulancejob`, `ak47_ambulancejob`, `tk_ambulancejob`, `p_ambulancejob`

## Banking

`auto`, `okokBanking`, `qb-banking`, `Renewed-Banking`, `qb-management`, `tgg-banking`, `qs_banking`, `fd_banking`

## Clothing

`auto`, `skinchanger`, `fivem-appearance`, `illenium-appearance`, `rcore_clothing`, `qb-clothing`, `crm-appearance`, `p_appearance`, `qs-appearance`

## Dispatch

`auto`, `cd_dispatch`, `ps-dispatch`, `qs-dispatch`, `tk_dispatch`, `rcore_dispatch`, `core_dispatch`, `wasabi_mdt`, `lb-tablet`

## Inventory

`auto`, `ox_inventory`, `ak47_inventory`, `qs-inventory`, `qb-inventory`, `lj-Inventory`, `ps-inventory`, `pappu-inventorynp`, `codem-inventory`

## Target

`auto`, `ox_target`, `qb-target`, `qtarget`

## Vehicle Keys

`auto`, `wasabi_carlock`, `ak47_vehiclekeys`, `qs-vehiclekeys`, `vehicles_keys`, `msk_vehiclekeys`, `Renewed-Vehiclekeys`, `qbx_vehiclekeys`, `qb-vehiclekeys`, `cd_garage`

## Vehicle Fuel

`auto`, `ox_fuel`, `msk_fuel`, `ti_fuel`, `TAM_Fuel`, `LegacyFuel`, `cdn-fuel`, `rcore_fuel`, `lyre_fuel`, `lc_fuel`

## Sounds

`auto`, `xsound`, `mx-surround`

## Phone

`auto`, `okokPhone`, `qb-phone`, `qs-smartphone`, `qs-smartphone-pro`, `lb-phone`, `gksphone`

## Notification

`auto`, `fs_bridge`, `ox_lib`, `esx_notify`, `es_extended`, `qb-core`

## Text UI

`auto`, `fs_bridge`, `ox_lib`, `esx_textui`, `es_extended`, `qb-core`

## Progressbar

`auto`, `ox_lib`, `esx_progressbar`, `qb-core`, `mythic_progbar`

## Notes

- `selected_key = 1` means auto-detect
- use override docs when your exact resource is unsupported
- do not add unsupported resources directly into locked bridge files for normal installs
