# Callbacks

Bridge callbacks let your resources await a return value across client and server.

Use the pages below for the exact public callback helpers on each side:

- `Client` for waiting on server callbacks and registering client callbacks.
- `Server` for waiting on client callbacks and registering server callbacks.
