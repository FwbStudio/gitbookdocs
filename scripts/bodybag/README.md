#### ⚙️ Key Features

* **🔫 Kill & Bag**\
  Kill a player and use a bodybag to transport their body — immersive, seamless, and RP-friendly.
* **🧰 Create Multiple Bodybag Items**
  * Make **custom bodybag types** in config.
  * Whitelist each item for **specific jobs** (e.g., EMS, Mafia, Police).
  * Configure each item to **permanently delete the character** or just simulate death.
* **🌍 Burial & Burning Locations**
  * Set up **multiple graveyard or cremation zones**.
  * Players must have required items (like a shovel or gasoline) to dispose of bodies.
* **⚰️ Burying Logic**
  * If **delete character** is enabled:
    * Player is **kicked** from the server.
    * Character is permanently deleted.
    * Server-wide **announcement** about the burial.
  * If **delete character** is disabled:
    * Player is sent to **Heaven or Hell** (random or chosen via config).
    * Fully customizable **config message**.
* **🔥 Burning Logic**
  * Requires an item (e.g., **fuel**).
  * Same behavior as burial (delete character or Heaven/Hell).
  * If **character isn’t deleted**, after some time the body **decomposes**, and you receive a **DeadOpp item to smoke**.
* **🌐 Multilingual Ready**
  * Includes **open and editable language files**.
  * Translate the entire script to **any language** with ease.
* **🏙️ Heaven & Hell MLOs Included**
  * Visual "afterlife" zones for full RP immersion.
  * Supports custom teleport with visuals and optional effects.
* **🩺 EMS Support**
  * Fully compatible with **Wasabi's stretcher**.
  * Carry bodies with bodybags and use them in coordinated EMS RP.
* **♻️ Auto Cleanup**
  * If a player with a bodybag is revived, the bag is **automatically removed** to prevent reuse.
