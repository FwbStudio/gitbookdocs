# Installation

### **Framework**

* [ESX](https://github.com/esx-framework/esx_core)
* [QBCore](https://github.com/qbcore-framework/qb-core)

### Dependencies

| Resource                                                                                   | Install  | Description                                                             |
| ------------------------------------------------------------------------------------------ | -------- | ----------------------------------------------------------------------- |
| [ox\_lib](https://github.com/overextended/ox_lib)                                          | Required | context                                                                 |
| [Bridge](../fs_bridge/README.md)                     | Required | callbacks and other stuffs                                              |
| [Clothing As Item](../clothing-as-item/README.md) | optional | only needed if you want to make useable clothes craft from skitz tables |

### 1 - Start

```
1. Download your resource from FiveM's Keymaster.
2. Unzip fs_skitz.zip folder and place it into your resource folder.
3. install item accoridng to your inventory.
4. install images accoridng to your inventory.
```

### 3 - Configuration

### 5 - Ready

Congratulations, you've successfully installed the resource. Restart the server and you will be all set.
