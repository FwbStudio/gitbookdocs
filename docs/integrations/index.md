# Integrations

This section turns the current compatibility information into smaller reference pages that are easier to maintain as support expands.

## Public Integration Rules

- keep `selected_key = 1` for auto-detect unless support tells you otherwise
- start supported resources before `fs_bridge`
- use manual overrides when your exact resource is unsupported
- keep your bridge configuration readable and close to the documented public flow
