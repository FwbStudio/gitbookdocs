# Phone

Bridge currently supports the following phone resources in the public compatibility layer.

| Resource | Status |
| --- | --- |
| `okokPhone` | Supported |
| `qb-phone` | Supported |
| `qs-smartphone` | Supported |
| `qs-smartphone-pro` | Supported |
| `lb-phone` | Supported |
| `gksphone` | Supported |

Phone-specific custom behavior is usually documented through manual compatibility on the server side.
