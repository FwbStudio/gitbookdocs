# Vehicle Keys

Bridge currently supports the following public vehicle key integrations.

| Resource | Status |
| --- | --- |
| `wasabi_carlock` | Supported |
| `ak47_vehiclekeys` | Supported |
| `qs-vehiclekeys` | Supported |
| `vehicles_keys` | Supported |
| `msk_vehiclekeys` | Supported |
| `Renewed-Vehiclekeys` | Supported |
| `qbx_vehiclekeys` | Supported |
| `qb-vehiclekeys` | Supported |
| `cd_garage` | Supported |

Use [Vehicles](/vehicles/) for the public API surface that sits on top of these integrations.
